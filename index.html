An Introduction to Frow

Frow is an HTML and CSS gridwork for creating quick, responsive, mobile first projects on the web.

What Frow Is:

Frow (flex row) is a grid system for CSS which utilizes and enhances the CSS display property: flex.

If you know the CSS property display:flex, then you are going to love Frow. If you don't know it yet, then you are going to love the CSS property display:flex, and thus, Frow.

Frow is open source and available on GitHub.

What Frow Isn't:

Frow isn't a "full" Bootstrap replacement, yet. While similar, Frow is more lightweight, being just a super-cool grid system with some very minor styling tools, sans bloat.

Frow also isn't finished. It is brand new and receiving updates all the time. Until version "1.0.0" is hit, consider this a beta.
